## What is this?

**Zynra Spawner Protection** is a Fabric client mod for **Minecraft** built around **DonutSMP-style** spawner play. It helps you **protect spawner setups** with automation and safety-focused behavior—so unexpected visitors near your farm are handled before theft or grief becomes likely.

### About this repository

This GitHub repo is for **documentation and pointers only**. It does **not** contain mod source code, and you **cannot** build or compile the mod from here. Official **downloads**, **updates**, and **licensing** are only through **[zynra.net](https://zynra.net)**.

---

## Who is it for?

- Players who run **spawner farms** on supported servers and want **extra protection** around their workflow.
- Anyone who wants **clear, in-game controls** for automation (start/stop, farm vs. orders-style flows where applicable) and **safety-oriented** responses when conditions look risky.

## Main ideas (plain language)

- **Whitelist** — You can mark trusted players so the mod does not treat them as a threat.
- **Admin-aware behavior** — Designed so normal admin presence does not cause the mod to destroy your spawners or trigger the wrong actions (always follow server rules).
- **Farm vs. advanced flow** — You can tune behavior for **drop-focused farming** or richer **orders-style** automation, depending on how you play.
- **Reconnect / session resilience** — If the client drops, the mod aims to **get you back in a controlled way** instead of leaving the farm in a bad state.
- **Maintenance-style awareness** — When the environment looks like “maintenance” or unusual server state, the mod can **back off** from hostile-player logic until things look normal again.

Exact behavior depends on **your settings**, **server rules**, and the **build** you run. Always follow DonutSMP (or your server) terms.

## Installation (typical)

1. Install **Fabric Loader** for the correct game version.
2. Put the mod **`.jar`** in your `.minecraft/mods` folder (or your launcher’s mods path).
3. Start the game, open the mod UI, enter your **license** if prompted, then configure spawners and options as needed.

For downloads, updates, and support channels, use **[https://zynra.net](https://zynra.net)**.

---

## Configuration

Settings are stored in a **config file** under your Minecraft config directory (name may match the mod branding). You can edit positions, trusted players, toggles for farm/orders behavior, and related options from the in-game menu where available.

**Tip:** Keep backups of your config if you run multiple accounts or reinstall the game.

## Security & trust

- Prefer **official** downloads and documentation from **[zynra.net](https://zynra.net)**.
- Do not use repacked jars from random file hosts; they can contain malware or stolen builds.
- This mod is a **client tool**. It cannot guarantee what other players’ clients do; it focuses on **your** session, **your** farm, and **your** configured safety behavior.
